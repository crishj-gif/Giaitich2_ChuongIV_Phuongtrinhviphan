Chương IV: Phương Trình Vi Phân

I. PHƯƠNG TRÌNH VI PHÂN CẤP 1

1. Dạng Tách Biến

Dạng: $f(x)dx + g(y)dy = 0$ hoặc $y' = f(x)g(y)$.

B1: Chuyển hết biến $x$ về một vế (kèm $dx$), biến $y$ về một vế (kèm $dy$).

B2: Tích phân hai vế: $\int f(x)dx + \int g(y)dy = C$.

B3: Tính tích phân và rút ra biểu thức nghiệm.

Ví dụ Nâng cao (Cần đổi biến):

Giải phương trình: $y' = \cos(x + y + 1)$

Xem lời giải chi tiết
Bước 1: Nhận dạng và Đặt ẩn phụ Không thể tách trực tiếp $x, y$. Đặt $u = x + y + 1 \Rightarrow u' = 1 + y' \Rightarrow y' = u' - 1$.
PT trở thành: $u' - 1 = \cos u \Rightarrow u' = 1 + \cos u = 2\cos^2(u/2)$.
Bước 2: Tách biến và Tích phân $$\frac{du}{2\cos^2(u/2)} = dx \Rightarrow \int \frac{1}{2} \sec^2(u/2) du = \int dx$$ Lưu ý: $\int \sec^2(at) dt = \frac{1}{a}\tan(at)$. Ở đây $a=1/2$, nên nguyên hàm là $\tan(u/2)$.
Bước 3: Kết luận $$\tan\left(\frac{x+y+1}{2}\right) = x + C$$

2. Dạng Thuần Nhất

Dạng: $y' = f\left(\frac{y}{x}\right)$.

B1: Đặt $u = \frac{y}{x} \Rightarrow y = ux \Rightarrow y' = u'x + u$.

B2: Thay vào phương trình ban đầu, ta được PTVP tách biến theo $u$ và $x$: $u'x = f(u) - u$.

B3: Giải PT tách biến: $\frac{du}{f(u)-u} = \frac{dx}{x}$.

B4: Trả lại biến cũ ($u = y/x$) và kết luận.

Ví dụ Nâng cao (Tích phân hàm vô tỉ):

Giải phương trình: $x y' = y + \sqrt{x^2 - y^2} \quad (x>0)$

Xem lời giải chi tiết
Bước 1: Biến đổi về dạng thuần nhất $$y' = \frac{y}{x} + \sqrt{1 - \left(\frac{y}{x}\right)^2}$$ Đặt $u = y/x \Rightarrow y' = u'x + u$.
Bước 2: Thay thế và Tách biến $$u'x + u = u + \sqrt{1-u^2} \Rightarrow u'x = \sqrt{1-u^2} \Rightarrow \frac{du}{\sqrt{1-u^2}} = \frac{dx}{x}$$
Bước 3: Tính tích phân khó $\int \frac{du}{\sqrt{1-u^2}} = \arcsin u$. $$\arcsin u = \ln|x| + C$$
Bước 4: Kết luận $$\arcsin\left(\frac{y}{x}\right) = \ln|x| + C \Rightarrow y = x \sin(\ln|x| + C)$$

3. Dạng Tuyến Tính Cấp 1

Dạng: $y' + p(x)y = q(x)$.

B1: Tìm thừa số tích phân: $\mu(x) = e^{\int p(x)dx}$.

B2: Nhân hai vế với $\mu(x)$, đưa VT về đạo hàm tích: $(\mu(x) \cdot y)' = \mu(x)q(x)$.

B3: Tích phân hai vế: $y = \frac{1}{\mu(x)}\left(\int \mu(x)q(x)dx + C\right)$.

Ví dụ Nâng cao (Đảo vai trò x và y):

Giải phương trình: $(2y + x e^y) y' = 1$

Xem lời giải chi tiết
Bước 1: Nhận dạng ẩn Nếu coi $y$ là hàm của $x$, PT rất phức tạp (phi tuyến). Hãy coi $x$ là hàm của $y$ ($x = x(y)$). $$y' = \frac{dy}{dx} \Rightarrow \frac{dx}{dy} = \frac{1}{y'} = 2y + x e^y$$ $$x' - e^y x = 2y$$ Đây là PT tuyến tính cấp 1 đối với hàm $x(y)$.
Bước 2: Tìm thừa số tích phân $p(y) = -e^y \Rightarrow \mu(y) = e^{\int -e^y dy} = e^{-e^y}$.
Bước 3: Giải PT tuyến tính $$x = \frac{1}{e^{-e^y}} \left( \int 2y e^{-e^y} dy + C \right)$$ (Lưu ý: Tích phân $\int 2y e^{-e^y} dy$ không tính được sơ cấp, có thể giữ nguyên dạng tích phân hoặc đề bài thường cho dạng tính được. Ví dụ sửa đề thành $x' - \frac{1}{y}x = 2y$ sẽ dễ hơn. Với ví dụ này, ta dừng ở bước nhận dạng đảo biến).
Xét bài toán khác tính được: $y' = \frac{1}{x \sin y + 2 \sin 2y} \Rightarrow x' - (\sin y)x = 2\sin 2y$.
$\mu(y) = e^{\cos y}$.

4. Dạng Bernoulli

Dạng: $y' + p(x)y = q(x)y^n \quad (n \ne 0, 1)$.

B1: Chia hai vế cho $y^n$: $y^{-n}y' + p(x)y^{1-n} = q(x)$.

B2: Đặt $z = y^{1-n} \Rightarrow z' = (1-n)y^{-n}y'$.

B3: Thay vào để được PT tuyến tính theo $z$: $\frac{z'}{1-n} + p(x)z = q(x)$.

B4: Giải PT tuyến tính tìm $z$, sau đó suy ra $y$.

Ví dụ Nâng cao (Bernoulli ẩn):

Giải phương trình: $x dy + y(1 - y \ln x) dx = 0$

Xem lời giải chi tiết
Bước 1: Chuyển dạng $$x y' + y - y^2 \ln x = 0 \Leftrightarrow y' + \frac{1}{x}y = \frac{\ln x}{x} y^2$$ Đây là Bernoulli với $n=2$.
Bước 2: Đặt ẩn phụ Chia $y^2$: $y^{-2}y' + \frac{1}{x}y^{-1} = \frac{\ln x}{x}$. Đặt $z = y^{-1} \Rightarrow z' = -y^{-2}y'$. $$-z' + \frac{1}{x}z = \frac{\ln x}{x} \Rightarrow z' - \frac{1}{x}z = -\frac{\ln x}{x}$$
Bước 3: Tích phân từng phần $\mu(x) = \frac{1}{x}$. PT: $(\frac{z}{x})' = -\frac{\ln x}{x^2}$. $$\frac{z}{x} = \int -\frac{\ln x}{x^2} dx$$ Đặt $u = \ln x, dv = -x^{-2}dx \Rightarrow du = dx/x, v = x^{-1}$. $\int = x^{-1}\ln x - \int x^{-2}dx = \frac{\ln x}{x} + \frac{1}{x} + C$.
Bước 4: Kết luận $z = \ln x + 1 + Cx \Rightarrow y = \frac{1}{\ln x + 1 + Cx}$.

5. Dạng Vi Phân Toàn Phần

Dạng: $P(x,y)dx + Q(x,y)dy = 0$ thỏa mãn $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$.

B1: Kiểm tra điều kiện: $P_y = Q_x$.

B2: Chọn tích phân: $u(x,y) = \int P(x,y)dx + \varphi(y)$.

B3: Đạo hàm $u$ theo $y$ rồi cho bằng $Q(x,y)$ để tìm $\varphi'(y)$, từ đó tìm $\varphi(y)$.

B4: Kết luận nghiệm tổng quát: $u(x,y) = C$.

Ví dụ Nâng cao (Đạo hàm phức tạp):

Giải phương trình: $(e^x \sin y + 3y)dx + (e^x \cos y + 3x - 2y)dy = 0$

Xem lời giải chi tiết
Bước 1: Kiểm tra $P = e^x \sin y + 3y \Rightarrow P_y = e^x \cos y + 3$. $Q = e^x \cos y + 3x - 2y \Rightarrow Q_x = e^x \cos y + 3$. $P_y = Q_x \Rightarrow$ Vi phân toàn phần.
Bước 2: Tìm hàm u $$u = \int P dx = \int (e^x \sin y + 3y) dx = e^x \sin y + 3xy + \varphi(y)$$
Bước 3: Xác định $\varphi(y)$ $u'_y = e^x \cos y + 3x + \varphi'(y)$. So sánh với $Q$: $e^x \cos y + 3x + \varphi'(y) = e^x \cos y + 3x - 2y$. $\Rightarrow \varphi'(y) = -2y \Rightarrow \varphi(y) = -y^2$.
Bước 4: Kết luận $e^x \sin y + 3xy - y^2 = C$.

6. Dạng Thừa Số Tích Phân

Dạng: $Pdx + Qdy = 0$ nhưng $P_y \ne Q_x$. Cần tìm nhân tử $\mu$ để đưa về dạng toàn phần.

B1: Tính $\Delta_1 = \frac{P_y - Q_x}{Q}$. Nếu $\Delta_1$ chỉ chứa $x$, $\mu(x) = e^{\int \Delta_1 dx}$.

B2: Nếu B1 không được, tính $\Delta_2 = \frac{Q_x - P_y}{P}$. Nếu $\Delta_2$ chỉ chứa $y$, $\mu(y) = e^{\int \Delta_2 dy}$.

B3: Nhân $\mu$ vào phương trình ban đầu.

B4: Giải theo phương pháp Vi phân toàn phần (Dạng 5).

II. PHƯƠNG TRÌNH VI PHÂN CẤP 2

1. Dạng Giảm Cấp

Loại 1: Vắng $y$ ($F(x, y', y'')=0$)

B1: Đặt $p(x) = y' \Rightarrow y'' = p'$.

B2: Giải PT cấp 1 tìm $p(x)$.

B3: Giải $y' = p(x)$ tìm $y$.

Loại 2: Vắng $x$ ($F(y, y', y'')=0$)

B1: Đặt $p(y) = y' \Rightarrow y'' = p \cdot p'$.

B2: Giải PT cấp 1 tìm $p(y)$.

B3: Giải $y' = p(y)$ (tách biến) tìm $y$.

Ví dụ Nâng cao (Vắng x - Biến đổi phức tạp):

Giải phương trình: $2yy'' - 3(y')^2 = 4y^2$

Xem lời giải chi tiết
Bước 1: Đặt ẩn phụ Đặt $y' = p(y) \Rightarrow y'' = p \cdot p'$. PT: $2y p p' - 3p^2 = 4y^2$. (Coi đây là PT Bernoulli đối với hàm $p(y)$ hoặc coi $p^2 = z$).
Bước 2: Giải tìm p Đặt $u = p^2 \Rightarrow u' = 2pp'$. $y u' - 3u = 4y^2 \Rightarrow u' - \frac{3}{y}u = 4y$. (Tuyến tính cấp 1) Giải ra: $u = C_1 y^3 - 4y^2$.
Bước 3: Giải tìm y (Tích phân khó) $p = \pm y \sqrt{C_1 y - 4} = \frac{dy}{dx}$. $$\frac{dy}{y \sqrt{C_1 y - 4}} = \pm dx$$ Đặt $t = \sqrt{C_1 y - 4}$ để tính tích phân vế trái.

2. Dạng Phương Trình Tuyến Tính (Tổng Quát)

Dạng: $y'' + p(x)y' + q(x)y = f(x)$.

B1: Giải PT thuần nhất tương ứng ($f(x)=0$) để tìm nghiệm tổng quát $y_h = C_1 y_1 + C_2 y_2$.

B2: Tìm một nghiệm riêng $y_p$ của PT không thuần nhất.

B3: Nghiệm tổng quát: $y = y_h + y_p$.

3. Dạng Biến Thiên Hằng Số (Lagrange)

Dùng cho mọi dạng $f(x)$ mà ta đã biết nghiệm thuần nhất.

B1: Tìm nghiệm thuần nhất: $y_h = C_1 y_1 + C_2 y_2$.

B2: Coi $C_1, C_2$ là hàm của $x$. Giải hệ phương trình:

$$ \begin{cases} C_1'(x) y_1 + C_2'(x) y_2 = 0 \\ C_1'(x) y_1' + C_2'(x) y_2' = f(x) \end{cases} $$

B3: Tìm $C_1(x), C_2(x)$ bằng tích phân.

B4: Thế vào công thức nghiệm: $y = C_1(x)y_1 + C_2(x)y_2$.

Ví dụ Nâng cao (Tích phân lượng giác khó):

Giải phương trình: $y'' + y = \sec^3 x$

Xem lời giải chi tiết
Bước 1: Nghiệm thuần nhất $y_h = C_1 \cos x + C_2 \sin x$.
Bước 2: Hệ phương trình Lagrange $$ \begin{cases} C_1' \cos x + C_2' \sin x = 0 \\ -C_1' \sin x + C_2' \cos x = \sec^3 x \end{cases} $$ $C_1' = -\sin x \sec^3 x = -\frac{\sin x}{\cos^3 x}$. $C_2' = \cos x \sec^3 x = \frac{1}{\cos^2 x} = \sec^2 x$.
Bước 3: Tính tích phân $C_1 = \int -\tan x \sec^2 x dx = -\frac{1}{2}\tan^2 x$. (hoặc $-1/2 \sec^2 x$). $C_2 = \int \sec^2 x dx = \tan x$.
Bước 4: Kết luận $y = y_h - \frac{1}{2}\tan^2 x \cos x + \tan x \sin x = y_h + \frac{1}{2}\sin x \tan x$.

4. Dạng Hệ Số Hằng (Phương pháp hệ số bất định)

Áp dụng khi hệ số là hằng số và vế phải $f(x)$ có dạng đặc biệt.

Bước 1: Giải phương trình đặc trưng $ak^2 + bk + c = 0$ tìm nghiệm $k_1, k_2$.

  • 2 nghiệm thực phân biệt $k_1, k_2 \Rightarrow y_h = C_1 e^{k_1 x} + C_2 e^{k_2 x}$.
  • Nghiệm kép $k_1 = k_2 = k \Rightarrow y_h = (C_1 + C_2 x)e^{kx}$.
  • Nghiệm phức $k = \alpha \pm i\beta \Rightarrow y_h = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$.

Bước 2: Chọn nghiệm riêng $y_p$ dựa vào dạng của $f(x)$ (Xem bảng dưới).

Bước 3: Đạo hàm $y_p$, thay vào phương trình gốc để đồng nhất hệ số tìm các tham số.

Bước 4: Kết luận $y = y_h + y_p$.

Dạng của $f(x)$ Kiểm tra nghiệm đặc trưng Dạng nghiệm riêng $y_p$ cần chọn
Dạng 1:
$P_n(x) e^{\alpha x}$
($P_n$ là đa thức bậc $n$)		 $\alpha$ không là nghiệm của PT đặc trưng $Q_n(x) e^{\alpha x}$
$\alpha$ là nghiệm đơn ($k_1=\alpha$ hoặc $k_2=\alpha$) $x \cdot Q_n(x) e^{\alpha x}$ (Nhân thêm x)
$\alpha$ là nghiệm kép ($k_1=k_2=\alpha$) $x^2 \cdot Q_n(x) e^{\alpha x}$ (Nhân thêm x²)
Dạng 2:
$e^{\alpha x} [P_n(x)\cos\beta x + Q_m(x)\sin\beta x]$		 $\alpha \pm i\beta$ không là nghiệm đặc trưng $e^{\alpha x} [A_N(x)\cos\beta x + B_N(x)\sin\beta x]$
($N = \max(n,m)$)
$\alpha \pm i\beta$ là nghiệm đặc trưng $x \cdot e^{\alpha x} [A_N(x)\cos\beta x + B_N(x)\sin\beta x]$ (Nhân thêm x)

Ví dụ A (Dạng mũ - cộng hưởng):

Giải $y'' - 2y' + y = e^x$

Lời giải

1. PT đặc trưng: $k^2 - 2k + 1 = 0 \Rightarrow k=1$ (kép).

2. $f(x) = 1 \cdot e^{1x}$. Ta có $\alpha = 1$ trùng nghiệm kép.

3. Chọn $y_p = A x^2 e^x$.

4. Tính $y_p', y_p''$ thay vào ta được $2A = 1 \Rightarrow A = 1/2$.

$\Rightarrow y = (C_1 + C_2 x)e^x + \frac{1}{2}x^2 e^x$.

Ví dụ B (Dạng lượng giác):

Giải $y'' + y = \cos x$

Lời giải

1. PT đặc trưng: $k^2 + 1 = 0 \Rightarrow k = \pm i$.

2. $f(x) = \cos x = e^{0x}\cos 1x$. Số $\alpha \pm i\beta = \pm i$ là nghiệm PT đặc trưng.

3. Chọn $y_p = x(A \cos x + B \sin x)$.

4. Thay vào giải ra $A=0, B=1/2$.

$\Rightarrow y = C_1 \cos x + C_2 \sin x + \frac{1}{2}x\sin x$.

Ví dụ Nâng cao (Đa thức nhân Lượng giác - Tính toán khó):

Giải phương trình: $y'' + y = x \sin x$

Xem lời giải chi tiết
Bước 1: Nghiệm thuần nhất $k = \pm i$. $y_h = C_1 \cos x + C_2 \sin x$.
Bước 2: Dự đoán nghiệm riêng (Cộng hưởng) $f(x) = P_1(x) \sin 1x$. $\alpha=0, \beta=1 \Rightarrow \pm i$ là nghiệm. Nhân thêm $x$: $y_p = x [ (Ax+B)\cos x + (Cx+D)\sin x ]$. Tức là $y_p = (Ax^2 + Bx)\cos x + (Cx^2 + Dx)\sin x$.
Bước 3: Đạo hàm và Đồng nhất hệ số Tính $y_p'', y_p''$ rất dài. Sau khi rút gọn và thay vào PT: $2A \cos x - 4Ax \sin x - 2B \sin x + 2C \sin x + 4Cx \cos x + 2D \cos x = x \sin x$. So sánh hệ số:
  • $x \sin x$: $-4A = 1 \Rightarrow A = -1/4$.
  • $x \cos x$: $4C = 0 \Rightarrow C = 0$.
  • $\sin x$: $-2B + 2C = 0 \Rightarrow B = 0$.
  • $\cos x$: $2A + 2D = 0 \Rightarrow D = -A = 1/4$.
Bước 4: Kết luận $y_p = -\frac{1}{4}x^2 \cos x + \frac{1}{4}x \sin x$.

III. HỆ PHƯƠNG TRÌNH VI PHÂN

1. Phương Pháp Khử

B1: Từ một PT trong hệ, rút một ẩn theo ẩn còn lại và đạo hàm của nó.

B2: Đạo hàm biểu thức vừa rút, sau đó thế vào PT còn lại để được một PT cấp cao chỉ chứa 1 ẩn.

B3: Giải PT cấp cao tìm ẩn thứ nhất.

B4: Thế ngược lại để tìm ẩn thứ hai.

Ví dụ Nâng cao (Hệ ma trận chéo hóa phức tạp):

Giải hệ: $\begin{cases} x' = x - 5y \\ y' = x - y \end{cases}$

Xem lời giải chi tiết
Bước 1: Khử $x' = x - 5y \Rightarrow 5y = x - x'$. $5y' = x' - x''$. Thế vào PT 2 (nhân 5): $x' - x'' = 5x - 5y = 5x - (x - x') = 4x + x'$.
Bước 2: Giải PT cấp 2 $x'' + 4x = 0 \Rightarrow k = \pm 2i$. $x(t) = C_1 \cos 2t + C_2 \sin 2t$.
Bước 3: Tìm y $x'(t) = -2C_1 \sin 2t + 2C_2 \cos 2t$. $y = \frac{1}{5}(x - x') = \frac{1}{5}[(C_1 - 2C_2)\cos 2t + (C_2 + 2C_1)\sin 2t]$. (Việc tính toán hệ số của $y$ khá dễ nhầm dấu).

IV. BÀI TẬP LUYỆN TẬP (Từ dễ đến khó)

Mức 1: PTVP Cấp 1 Cơ Bản (Tách biến, Tuyến tính, Thuần nhất)

Bài 1: $y' = 3x^2 y$

Lời giải

Dạng: Tách biến

1. $\frac{dy}{y} = 3x^2 dx$.

2. $\int \frac{dy}{y} = \int 3x^2 dx \Rightarrow \ln|y| = x^3 + C$.

3. $y = Ce^{x^3}$.

Bài 2: $y' + \frac{2}{x}y = 0$

Lời giải

Dạng: Tách biến (hoặc Tuyến tính thuần nhất)

1. $\frac{dy}{y} = -\frac{2}{x}dx$.

2. $\ln|y| = -2\ln|x| + \ln|C| = \ln|Cx^{-2}|$.

3. $y = \frac{C}{x^2}$.

Bài 3: $y' + y = 2x$

Lời giải

Dạng: Tuyến tính cấp 1

1. $p(x)=1 \Rightarrow \mu(x) = e^x$.

2. $(y e^x)' = 2x e^x$.

3. Tích phân từng phần: $y e^x = \int 2x e^x dx = 2(xe^x - e^x) + C$.

4. $y = 2(x-1) + Ce^{-x}$.

Bài 4: $(x^2 + y^2)dx - xy dy = 0$

Lời giải

Dạng: Thuần nhất

1. $y' = \frac{x^2+y^2}{xy} = \frac{x}{y} + \frac{y}{x}$. Đặt $u = y/x$.

2. $u'x + u = \frac{1}{u} + u \Rightarrow u'x = \frac{1}{u}$.

3. $u du = \frac{dx}{x} \Rightarrow \frac{u^2}{2} = \ln|x| + C$.

4. $\frac{y^2}{2x^2} = \ln|x| + C \Rightarrow y^2 = 2x^2(\ln|x| + C)$.

Mức 2: Trung Bình (Bernoulli, VP Toàn phần, Cấp 2 cơ bản)

Bài 5: $y' + \frac{y}{x} = y^2$

Lời giải

Dạng: Bernoulli ($n=2$)

1. Chia $y^2$: $y^{-2}y' + \frac{1}{x}y^{-1} = 1$. Đặt $z = y^{-1}$.

2. $-z' + \frac{1}{x}z = 1 \Rightarrow z' - \frac{1}{x}z = -1$.

3. Giải tuyến tính $z$: $\mu = 1/x \Rightarrow (\frac{z}{x})' = -\frac{1}{x}$.

4. $z = x(C - \ln|x|) \Rightarrow y = [x(C-\ln|x|)]^{-1}$.

Bài 6: $(2x+y)dx + (x+2y)dy = 0$

Lời giải

Dạng: Vi phân toàn phần

1. $P_y = 1, Q_x = 1$ (khớp).

2. $u = \int (2x+y)dx = x^2 + xy + \varphi(y)$.

3. $u'_y = x + \varphi' = x + 2y \Rightarrow \varphi = y^2$.

4. Nghiệm: $x^2 + xy + y^2 = C$.

Bài 7: $(x^2+y)dx - xdy = 0$

Lời giải

Dạng: Thừa số tích phân

1. $(P_y - Q_x)/Q = (1 - (-1))/(-x) = -2/x$.

2. $\mu(x) = e^{\int -2/x dx} = 1/x^2$.

3. Nhân vào: $(1 + y/x^2)dx - (1/x)dy = 0$.

4. Giải VPTP: $x - y/x = C \Rightarrow y = x^2 - Cx$.

Bài 8: $y'' - y' = 0$

Lời giải

Dạng: Cấp 2 giảm cấp (Vắng y)

1. Đặt $p=y' \Rightarrow p' - p = 0 \Rightarrow p = C_1 e^x$.

2. $y' = C_1 e^x \Rightarrow y = C_1 e^x + C_2$.

Bài 9: $y y'' - (y')^2 = 0$

Lời giải

Dạng: Cấp 2 giảm cấp (Vắng x)

1. Đặt $y' = p(y) \Rightarrow y'' = p \cdot p'$. PT: $y p p' - p^2 = 0$.

2. $p(y p' - p) = 0$. Nếu $p \ne 0 \Rightarrow yp' = p \Rightarrow p = C_1 y$.

3. $y' = C_1 y \Rightarrow \ln|y| = C_1 x + C_2 \Rightarrow y = Ae^{Bx}$.

Bài 10: $y'' - 5y' + 6y = 0$

Lời giải

Dạng: Cấp 2 hệ số hằng thuần nhất

1. $k^2 - 5k + 6 = 0 \Rightarrow k_1 = 2, k_2 = 3$.

2. Nghiệm tổng quát: $y = C_1 e^{2x} + C_2 e^{3x}$.

Bài 11: $y'' - 4y' + 4y = e^{2x}$

Lời giải

Dạng: Hệ số bất định (Cộng hưởng kép)

1. $k^2 - 4k + 4 = 0 \Rightarrow k=2$ (kép).

2. $f(x) = e^{2x}$. $\alpha=2$ trùng nghiệm kép $\Rightarrow y_p = A x^2 e^{2x}$.

3. Tính đạo hàm thay vào: $2A = 1 \Rightarrow A = 1/2$.

4. KQ: $y = (C_1 + C_2 x)e^{2x} + \frac{1}{2}x^2 e^{2x}$.

Bài 12: $y'' + 9y = 5\cos 3x$

Lời giải

Dạng: Hệ số bất định (Lượng giác - Cộng hưởng)

1. $k^2 + 9 = 0 \Rightarrow k = \pm 3i$.

2. $\alpha=0, \beta=3 \Rightarrow 0 \pm 3i$ là nghiệm. $y_p = x(A \cos 3x + B \sin 3x)$.

3. Thay vào: $-6A \sin 3x + 6B \cos 3x = 5 \cos 3x$.

4. $A=0, B=5/6$. KQ: $y = C_1 \cos 3x + C_2 \sin 3x + \frac{5}{6}x \sin 3x$.

Bài 13: $y'' - 2y' + 2y = 2e^x \cos x$

Lời giải

Dạng: Hệ số bất định (Phức hợp)

1. $k^2 - 2k + 2 = 0 \Rightarrow k = 1 \pm i$.

2. $f(x)$ có $\alpha=1, \beta=1$. $1 \pm i$ là nghiệm. $y_p = x e^x (A \cos x + B \sin x)$.

3. Đồng nhất hệ số: $A=0, B=1$.

4. $y = e^x(C_1 \cos x + C_2 \sin x) + x e^x \sin x$.

Bài 14: $y'' - 3y' + 2y = 2x^2 + 1$

Lời giải

Dạng: Hệ số bất định (Đa thức)

1. $k=1, k=2$.

2. $f(x)$ đa thức bậc 2, $\alpha=0$ không là nghiệm. $y_p = Ax^2 + Bx + C$.

3. Thay vào: $2Ax^2 + (2B-6A)x + (2A-3B+2C) = 2x^2 + 1$.

4. Giải hệ: $A=1, B=3, C=7/2$.

Bài 15: $y'' + y = \frac{1}{\cos x}$

Lời giải

Dạng: Biến thiên hằng số (Lagrange)

1. $y_h = C_1 \cos x + C_2 \sin x$.

2. Wronskian $W = 1$. $C_1' = -\sin x / \cos x, C_2' = 1$.

3. $C_1 = \ln|\cos x|, C_2 = x$.

4. $y = \cos x \ln|\cos x| + x \sin x + y_h$.

Bài 16: $y'' - 3y' + 2y = e^x + e^{2x}$

Lời giải

Dạng: Nguyên lý chồng chất nghiệm

1. $y_h = C_1 e^x + C_2 e^{2x}$.

2. Tách $f_1 = e^x$ (nghiệm $k=1$) $\Rightarrow y_{p1} = -x e^x$.

3. Tách $f_2 = e^{2x}$ (nghiệm $k=2$) $\Rightarrow y_{p2} = x e^{2x}$.

4. $y = y_h - x e^x + x e^{2x}$.

Bài 17: $\begin{cases} x' = x - y \\ y' = x + 3y \end{cases}$

Lời giải

Dạng: Hệ PTVP (Khử)

1. $y = x - x'$. Thế vào (2): $x - x'' = x + 3(x-x')$.

2. $x'' - 4x' + 4x = 0 \Rightarrow x = (C_1 + C_2 t)e^{2t}$.

3. Tính $x'$ và thay vào tìm $y$.

Bài 18: $\frac{dx}{xz} = \frac{dy}{yz} = \frac{dz}{-xy}$

Lời giải

Dạng: Hệ đối xứng (Tích phân đầu)

1. $\frac{dx}{x} = \frac{dy}{y} \Rightarrow y = C_1 x$.

2. $x dx + y dy = \frac{x^2 z + y^2 z}{z} \dots$ không khả thi. Chọn cách khác.

3. $x dx + y dy + 2z dz = 0$ (nhân chéo kiểm tra lại). Hoặc từ $y=C_1 x$ thế vào.

Kết quả: $x/y = C_1$ và $xy + z^2 = C_2$.

Bài 19: $x^2 y'' - 2xy' + 2y = 0$

Lời giải

Dạng: Phương trình Euler

1. Đặt $x = e^t$. PT trở thành hệ số hằng: $\frac{d^2y}{dt^2} - 3\frac{dy}{dt} + 2y = 0$.

2. $k^2 - 3k + 2 = 0 \Rightarrow k=1, 2$.

3. $y = C_1 e^t + C_2 e^{2t} = C_1 x + C_2 x^2$.

Bài 20: Giải hệ ma trận $X' = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}X$

Lời giải

Dạng: Hệ ma trận (Trị riêng phức)

1. $\lambda^2 + 1 = 0 \Rightarrow \lambda = \pm i$.

2. Vectơ riêng $v_1 = (1, i)^T$.

3. Nghiệm thực: $X = C_1 \begin{pmatrix} \cos t \\ -\sin t \end{pmatrix} + C_2 \begin{pmatrix} \sin t \\ \cos t \end{pmatrix}$.
